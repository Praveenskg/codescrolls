# Python Data Structures

Python provides built-in data structures that cover most programming needs.

## List

```python
items = ["html", "css", "python"]
items.append("react")
```

## Tuple

```python
point = (10, 20)
```

## Dictionary

```python
user = {"name": "Praveen", "role": "developer"}
print(user["name"])
```

## Set

```python
tags = {"python", "docs", "python"}
print(tags)  # duplicates removed
```
