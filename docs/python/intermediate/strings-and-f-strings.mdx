# Python Strings and f-Strings

Strings are immutable sequences used for text operations.

## Common String Operations

```python
text = "  Python Docs  "
print(text.strip())
print(text.lower())
print(text.replace("Docs", "Guide"))
```

## Slicing

```python
word = "python"
print(word[0:3])  # pyt
print(word[-1])   # n
```

## f-Strings

```python
name = "Praveen"
score = 95
print(f"{name} scored {score}%")
```

f-strings are preferred for clarity and performance.

## Useful Methods

- `split()`
- `join()`
- `startswith()` and `endswith()`
- `find()`
