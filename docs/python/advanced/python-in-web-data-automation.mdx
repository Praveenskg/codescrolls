# Python in Web, Data, and Automation

Python is versatile and powers many software domains.

## Web Development

- Flask and Django for web apps
- FastAPI for modern async APIs
- Background jobs with Celery/RQ

## Data Engineering and Analysis

- NumPy and pandas for data processing
- ETL pipelines and scheduled batch jobs
- Data validation and transformation scripts

## AI and Machine Learning

- scikit-learn for classical ML
- PyTorch/TensorFlow for deep learning
- Model serving with FastAPI or Flask

## Automation and DevOps

- File, system, and browser automation
- API integration scripts
- CI/CD utilities and deployment tooling

## Career Relevance

Python skills transfer well across backend, analytics, platform engineering, and
AI product teams.
