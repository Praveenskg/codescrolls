# OOP in Python

Object-oriented programming models behavior and data together.

## Class and Object

```python
class User:
    def __init__(self, name: str):
        self.name = name

    def greet(self) -> str:
        return f"Hello, {self.name}"
```

## Inheritance

```python
class Admin(User):
    def can_manage(self) -> bool:
        return True
```

## Encapsulation

Use conventions like `_internal` for protected members and properties for
controlled access.

## Composition Over Inheritance

Prefer composition when reusing behavior without tight coupling.
