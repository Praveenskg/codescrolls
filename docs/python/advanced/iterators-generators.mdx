# Iterators and Generators in Python

Generators allow you to produce values lazily, one at a time.

## Iterator Protocol

An iterator implements:

- `__iter__()`
- `__next__()`

## Generator Function

```python
def countdown(n):
    while n > 0:
        yield n
        n -= 1

for value in countdown(3):
    print(value)
```

## Generator Expressions

```python
squares = (n * n for n in range(5))
```

Use generators for large data streams to reduce memory usage.
