# Python Control Flow

Control flow determines which code runs based on conditions or repetition.

## if / elif / else

```python
score = 82

if score >= 90:
    grade = "A"
elif score >= 75:
    grade = "B"
else:
    grade = "C"
```

## for Loop

```python
for i in range(1, 4):
    print(i)
```

## while Loop

```python
count = 3
while count > 0:
    print(count)
    count -= 1
```

## Loop Control

- `break`: stop the loop
- `continue`: skip current iteration
- `pass`: placeholder statement
