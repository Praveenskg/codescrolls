# Python Variables and Data Types

Python variables are dynamically typed and do not require explicit declarations.

## Variable Basics

```python
name = "CodeScrolls"
age = 2
is_active = True
```

## Core Data Types

- `str`: text values
- `int`: whole numbers
- `float`: decimal numbers
- `bool`: `True` or `False`
- `list`: ordered mutable collection
- `tuple`: ordered immutable collection
- `dict`: key-value mapping
- `set`: unordered unique values

## Type Checking

```python
value = 42
print(type(value))  # <class 'int'>
```

## Type Conversion

```python
num = int("10")
text = str(100)
price = float("19.99")
```
