# Python Functions

Functions package reusable logic into a named block.

## Defining a Function

```python
def greet(name):
    return f"Hello, {name}!"

print(greet("Praveen"))
```

## Parameters and Arguments

- Positional arguments
- Keyword arguments
- Default arguments

```python
def power(base, exp=2):
    return base ** exp

print(power(4))
print(power(base=2, exp=3))
```

## Variable-Length Arguments

```python
def total(*numbers):
    return sum(numbers)

def build_profile(**details):
    return details
```

## Return Values

Functions return values with `return`. If omitted, Python returns `None`.

## Type Hints in Functions

```python
def add(a: int, b: int) -> int:
    return a + b
```

## Best Practices

- Keep each function focused on one task
- Use descriptive names
- Avoid hidden side effects
- Add docstrings for public APIs

## Example with Docstring

```python
def to_slug(text: str) -> str:
    """Convert text to a URL-friendly slug."""
    return text.strip().lower().replace(" ", "-")
```
