# Python Syntax and Indentation

Python uses indentation instead of braces to define code blocks.

## Why It Matters

Indentation is part of Python syntax. Wrong indentation causes
`IndentationError`.

## Standard Rule

Use 4 spaces per indentation level.

```python
if True:
    print("Valid block")
```

## Common Mistakes

- Mixing tabs and spaces
- Inconsistent nesting depth
- Misaligned blocks after copy-paste

## Editor Tip

Enable "insert spaces" and "format on save" in your editor.
