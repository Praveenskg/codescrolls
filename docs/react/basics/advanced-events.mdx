## âŒ¨ï¸ Keyboard Events

React provides events for key presses, such as `onKeyDown`, `onKeyUp`, and `onKeyPress`.

```jsx live
function KeyPressExample() {
  const handleKeyPress = (e) => {
    if (e.key === 'Enter') {
      alert('Enter key pressed!');
    }
  };

  return <input type='text' onKeyDown={handleKeyPress} placeholder='Press Enter...' />;
}
```

## ðŸŒ Event Propagation (Bubbling & Capturing)

React events bubble by default. You can **stop propagation** using `e.stopPropagation()`.

```jsx live
function PropagationExample() {
  const handleOuterClick = () => alert('Outer clicked');
  const handleInnerClick = (e) => {
    e.stopPropagation();
    alert('Inner clicked');
  };

  return (
    <div onClick={handleOuterClick} style={{ padding: '20px', background: '#eee' }}>
      <button onClick={handleInnerClick}>Click Inner</button>
    </div>
  );
}
```
