# React Events

React uses a **synthetic event system** to handle DOM events across all
browsers, making your event handling consistent and reliable.

Just like in HTML, you can use events like `onClick`, `onChange`, `onSubmit`,
etc., but with **camelCase** in JSX.

## ğŸ“¬ Form Submit with onSubmit

```jsx live
function ContactForm() {
  const [email, setEmail] = React.useState('');

  function handleSubmit(e) {
    e.preventDefault();
    alert(`Thanks for subscribing, ${email}`);
  }

  return (
    <form onSubmit={handleSubmit}>
      <input
        type="email"
        placeholder="Enter email"
        value={email}
        onChange={e => setEmail(e.target.value)}
      />
      <button type="submit">Subscribe</button>
    </form>
  );
}
```
