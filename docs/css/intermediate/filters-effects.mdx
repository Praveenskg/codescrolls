# ‚ú® CSS Filters & Effects

CSS filters apply visual effects to elements like blur, brightness, contrast,
and more. Create stunning designs without images or JavaScript!

## üìñ What are Filters?

Filters modify the rendering of an element before it's displayed, allowing you
to apply graphical effects like blur, color shifting, and more.

```css
.element {
  filter: blur(5px);
  filter: brightness(150%);
  filter: contrast(200%);
}
```

---

## üé® Filter Functions

### blur()

Applies a Gaussian blur effect.

```css
/* Blur values */
.element {
  filter: blur(0); /* No blur */
  filter: blur(5px); /* Medium blur */
  filter: blur(10px); /* Strong blur */
}
```

**Practical Examples:**

```css
/* Blurred background */
.background {
  filter: blur(8px);
}

/* Focus effect */
.gallery img {
  filter: blur(2px);
  transition: filter 0.3s;
}

.gallery img:hover {
  filter: blur(0);
}

/* Loading state */
.loading {
  filter: blur(3px);
  pointer-events: none;
}
```

### brightness()

Adjusts the brightness of an element.

```css
/* Brightness values */
.element {
  filter: brightness(100%); /* Normal (default) */
  filter: brightness(50%); /* Darker */
  filter: brightness(150%); /* Brighter */
  filter: brightness(200%); /* Very bright */
}
```

**Practical Examples:**

```css
/* Darken on hover */
.image {
  filter: brightness(100%);
  transition: filter 0.3s;
}

.image:hover {
  filter: brightness(70%);
}

/* Highlight effect */
.spotlight {
  filter: brightness(150%);
}

/* Night mode images */
.dark-mode img {
  filter: brightness(80%);
}
```

### contrast()

Adjusts the contrast of an element.

```css
/* Contrast values */
.element {
  filter: contrast(100%); /* Normal (default) */
  filter: contrast(50%); /* Less contrast */
  filter: contrast(200%); /* More contrast */
}
```

**Practical Examples:**

```css
/* High contrast mode */
.high-contrast {
  filter: contrast(150%);
}

/* Washed out effect */
.faded {
  filter: contrast(70%);
}
```

### grayscale()

Converts element to grayscale.

```css
/* Grayscale values */
.element {
  filter: grayscale(0%); /* Color (default) */
  filter: grayscale(50%); /* Partial grayscale */
  filter: grayscale(100%); /* Full grayscale */
}
```

**Practical Examples:**

```css
/* Inactive items */
.disabled {
  filter: grayscale(100%);
  opacity: 0.6;
}

/* Hover to color */
.thumbnail {
  filter: grayscale(100%);
  transition: filter 0.3s;
}

.thumbnail:hover {
  filter: grayscale(0%);
}

/* Archive indicator */
.archived {
  filter: grayscale(100%);
}
```

### hue-rotate()

Rotates the hue of an element.

```css
/* Hue rotation (0-360deg) */
.element {
  filter: hue-rotate(0deg); /* No change */
  filter: hue-rotate(90deg); /* 90¬∞ rotation */
  filter: hue-rotate(180deg); /* 180¬∞ rotation */
  filter: hue-rotate(360deg); /* Full circle = original */
}
```

**Practical Examples:**

```css
/* Color theme switcher */
.theme-blue {
  filter: hue-rotate(0deg);
}
.theme-purple {
  filter: hue-rotate(60deg);
}
.theme-red {
  filter: hue-rotate(120deg);
}

/* Animated rainbow effect */
@keyframes rainbow {
  from {
    filter: hue-rotate(0deg);
  }
  to {
    filter: hue-rotate(360deg);
  }
}

.rainbow {
  animation: rainbow 3s linear infinite;
}
```

### saturate()

Adjusts color saturation.

```css
/* Saturation values */
.element {
  filter: saturate(100%); /* Normal (default) */
  filter: saturate(0%); /* Grayscale */
  filter: saturate(50%); /* Desaturated */
  filter: saturate(200%); /* Highly saturated */
}
```

**Practical Examples:**

```css
/* Vibrant hover effect */
.image {
  filter: saturate(100%);
  transition: filter 0.3s;
}

.image:hover {
  filter: saturate(150%);
}

/* Muted background */
.background-image {
  filter: saturate(50%);
}
```

### invert()

Inverts colors of an element.

```css
/* Invert values */
.element {
  filter: invert(0%); /* Normal (default) */
  filter: invert(50%); /* 50% inverted */
  filter: invert(100%); /* Fully inverted */
}
```

**Practical Examples:**

```css
/* Dark mode toggle */
.dark-mode {
  filter: invert(100%);
}

/* Negative image effect */
.negative {
  filter: invert(100%);
}

/* Logo color adaptation */
.logo-dark {
  filter: invert(100%);
}
```

### opacity()

Adjusts transparency (similar to opacity property).

```css
/* Opacity values */
.element {
  filter: opacity(100%); /* Fully opaque */
  filter: opacity(50%); /* Semi-transparent */
  filter: opacity(0%); /* Fully transparent */
}
```

### sepia()

Applies a sepia tone effect.

```css
/* Sepia values */
.element {
  filter: sepia(0%); /* No sepia */
  filter: sepia(50%); /* Partial sepia */
  filter: sepia(100%); /* Full sepia */
}
```

**Practical Examples:**

```css
/* Vintage photo effect */
.vintage {
  filter: sepia(80%);
}

/* Old newspaper look */
.retro {
  filter: sepia(100%) contrast(120%);
}
```

### drop-shadow()

Adds a shadow to an element (respects transparency).

```css
/* Drop shadow syntax */
.element {
  filter: drop-shadow(offset-x offset-y blur-radius color);
}

/* Examples */
.element {
  filter: drop-shadow(2px 2px 4px rgba(0, 0, 0, 0.5));
  filter: drop-shadow(0 0 10px rgba(255, 0, 0, 0.8));
}
```

**Practical Examples:**

```css
/* Icon shadow (respects transparency) */
.icon {
  filter: drop-shadow(0 2px 4px rgba(0, 0, 0, 0.3));
}

/* Glow effect */
.glow {
  filter: drop-shadow(0 0 10px rgba(59, 130, 246, 0.8));
}

/* Neon text */
.neon {
  filter: drop-shadow(0 0 5px #fff) drop-shadow(0 0 10px #fff)
    drop-shadow(0 0 20px #ff00ff);
}
```

## üîó Combining Filters

Multiple filters are applied in sequence.

```css
/* Multiple filters */
.element {
  filter: contrast(150%) brightness(110%) saturate(120%);
}

/* Complex effects */
.vintage-photo {
  filter: sepia(60%) contrast(120%) brightness(90%) saturate(80%);
}

/* Dramatic effect */
.dramatic {
  filter: contrast(200%) brightness(80%) saturate(150%) blur(0.5px);
}
```

**Practical Examples:**

```css
/* Instagram-style filters */
.filter-nashville {
  filter: sepia(20%) contrast(120%) brightness(105%) saturate(120%);
}

.filter-toaster {
  filter: contrast(150%) brightness(90%) sepia(50%);
}

.filter-walden {
  filter: brightness(110%) hue-rotate(-10deg) sepia(30%) saturate(160%);
}
```

## üå´Ô∏è Backdrop Filter

Applies filters to the area **behind** an element (glassmorphism effect).

```css
/* Basic backdrop filter */
.glass {
  backdrop-filter: blur(10px);
  background-color: rgba(255, 255, 255, 0.1);
}
```

**Practical Examples:**

```css
/* Glass morphism card */
.glass-card {
  background: rgba(255, 255, 255, 0.1);
  backdrop-filter: blur(10px) saturate(180%);
  border: 1px solid rgba(255, 255, 255, 0.2);
  border-radius: 12px;
  padding: 2rem;
  box-shadow: 0 8px 32px rgba(0, 0, 0, 0.1);
}

/* Frosted glass navigation */
.nav {
  background: rgba(255, 255, 255, 0.8);
  backdrop-filter: blur(20px) saturate(180%);
  border-bottom: 1px solid rgba(255, 255, 255, 0.3);
}

/* Modal overlay */
.modal-overlay {
  background: rgba(0, 0, 0, 0.5);
  backdrop-filter: blur(5px);
}
```

## üé® Blend Modes

### mix-blend-mode

Controls how element blends with elements behind it.

```css
.element {
  mix-blend-mode: normal; /* Default */
  mix-blend-mode: multiply;
  mix-blend-mode: screen;
  mix-blend-mode: overlay;
  mix-blend-mode: darken;
  mix-blend-mode: lighten;
  mix-blend-mode: color-dodge;
  mix-blend-mode: color-burn;
  mix-blend-mode: hard-light;
  mix-blend-mode: soft-light;
  mix-blend-mode: difference;
  mix-blend-mode: exclusion;
  mix-blend-mode: hue;
  mix-blend-mode: saturation;
  mix-blend-mode: color;
  mix-blend-mode: luminosity;
}
```

**Practical Examples:**

```css
/* Text over image */
.hero-text {
  color: white;
  mix-blend-mode: difference;
}

/* Colorful overlays */
.overlay {
  background-color: #ff00ff;
  mix-blend-mode: multiply;
}

/* Duotone effect */
.duotone {
  position: relative;
}

.duotone::after {
  content: '';
  position: absolute;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
  background: linear-gradient(45deg, #667eea 0%, #764ba2 100%);
  mix-blend-mode: color;
}
```

### background-blend-mode

Controls how background images blend together.

```css
.element {
  background-image: url('texture.png'), linear-gradient(#667eea, #764ba2);
  background-blend-mode: multiply;
}
```

**Practical Examples:**

```css
/* Gradient overlay on image */
.hero {
  background-image:
    linear-gradient(rgba(0, 0, 0, 0.5), rgba(0, 0, 0, 0.5)), url('hero.jpg');
  background-blend-mode: normal;
  background-size: cover;
  background-position: center;
}

/* Texture overlay */
.textured {
  background-image: url('paper-texture.png'), linear-gradient(#fff, #f0f0f0);
  background-blend-mode: multiply;
}
```

## ‚úÖ Best Practices

1. **Use filters for hover effects**

   ```css
   img {
     filter: brightness(100%);
     transition: filter 0.3s;
   }

   img:hover {
     filter: brightness(110%) contrast(110%);
   }
   ```

2. **Combine filters thoughtfully**

   ```css
   /* Good: Subtle enhancement */
   .image {
     filter: contrast(105%) saturate(110%);
   }

   /* Avoid: Too many filters */
   .image {
     filter: blur(1px) brightness(110%) contrast(120%) saturate(150%)
       hue-rotate(10deg);
   }
   ```

3. **Consider performance**

   ```css
   /* Better: Use will-change for animated filters */
   .animated {
     will-change: filter;
     filter: blur(0);
     transition: filter 0.3s;
   }
   ```

4. **Browser compatibility**
   ```css
   /* Provide fallbacks */
   .glass {
     background: rgba(255, 255, 255, 0.8); /* Fallback */
     backdrop-filter: blur(10px);
   }
   ```

## ‚ö†Ô∏è Common Pitfalls

### ‚ùå Overusing Filters

```css
/* Bad: Performance impact */
.element {
  filter: blur(10px) brightness(150%) contrast(200%) saturate(300%);
}

/* Good: Minimal filters */
.element {
  filter: brightness(110%) contrast(110%);
}
```

### ‚ùå Backdrop Filter Browser Support

```css
/* Bad: No fallback */
.element {
  backdrop-filter: blur(10px);
}

/* Good: With fallback */
.element {
  background: rgba(255, 255, 255, 0.8);
}

@supports (backdrop-filter: blur(10px)) {
  .element {
    background: rgba(255, 255, 255, 0.3);
    backdrop-filter: blur(10px);
  }
}
```
