# üîÑ CSS Transforms

CSS transforms allow you to rotate, scale, skew, or translate elements in 2D and
3D space, creating dynamic and engaging visual effects without JavaScript.

## üìñ What are Transforms?

Transforms modify the coordinate space of an element, changing its position,
size, rotation, or shape.

```css
.element {
  transform: rotate(45deg);
  transform: scale(1.5);
  transform: translate(50px, 100px);
}
```

---

## üéØ 2D Transform Functions

### translate()

Move an element from its current position.

```css
/* Move right and down */
.box {
  transform: translate(50px, 100px);
}

/* Individual axes */
.box {
  transform: translateX(50px); /* Horizontal only */
  transform: translateY(100px); /* Vertical only */
}

/* Percentage values (relative to element size) */
.box {
  transform: translate(50%, 50%);
}
```

**Practical Examples:**

```css
/* Center an absolutely positioned element */
.modal {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
}

/* Slide-in animation */
.slide-in {
  transform: translateX(-100%);
  transition: transform 0.3s ease;
}

.slide-in.active {
  transform: translateX(0);
}
```

### scale()

Resize an element larger or smaller.

```css
/* Uniform scaling */
.box {
  transform: scale(1.5); /* 150% size */
  transform: scale(0.5); /* 50% size */
}

/* Different horizontal and vertical scaling */
.box {
  transform: scale(2, 0.5); /* 200% width, 50% height */
}

/* Individual axes */
.box {
  transform: scaleX(1.5); /* Horizontal only */
  transform: scaleY(0.8); /* Vertical only */
}
```

**Practical Examples:**

```css
/* Hover zoom effect */
.card {
  transition: transform 0.3s ease;
}

.card:hover {
  transform: scale(1.05);
}

/* Button press effect */
.button:active {
  transform: scale(0.95);
}

/* Flip effect (negative scale) */
.flip {
  transform: scaleX(-1); /* Horizontal flip */
}
```

### rotate()

Rotate an element around its transform origin.

```css
/* Degrees */
.box {
  transform: rotate(45deg);
  transform: rotate(-45deg); /* Counter-clockwise */
}

/* Other units */
.box {
  transform: rotate(0.25turn); /* Quarter turn */
  transform: rotate(3.14rad); /* Radians */
}
```

**Practical Examples:**

```css
/* Loading spinner */
.spinner {
  animation: spin 1s linear infinite;
}

@keyframes spin {
  from {
    transform: rotate(0deg);
  }
  to {
    transform: rotate(360deg);
  }
}

/* Tilted badge */
.badge {
  transform: rotate(-15deg);
}

/* Chevron icon */
.chevron {
  transform: rotate(90deg);
  transition: transform 0.3s;
}

.chevron.open {
  transform: rotate(-90deg);
}
```

### skew()

Skew or slant an element.

```css
/* Both axes */
.box {
  transform: skew(20deg, 10deg); /* X-axis, Y-axis */
}

/* Individual axes */
.box {
  transform: skewX(20deg); /* Horizontal skew */
  transform: skewY(10deg); /* Vertical skew */
}
```

**Practical Examples:**

```css
/* Parallelogram effect */
.slanted {
  transform: skewX(-15deg);
}

/* Perspective text effect */
.perspective-text {
  transform: skewY(-5deg);
}
```

## üîó Combining Transforms

Multiple transforms are applied in order (right to left).

```css
/* Multiple transforms */
.box {
  transform: translate(50px, 50px) rotate(45deg) scale(1.5);
}

/* Order matters! */
.box1 {
  transform: translate(100px, 0) rotate(45deg);
  /* Moves right, then rotates */
}

.box2 {
  transform: rotate(45deg) translate(100px, 0);
  /* Rotates, then moves diagonally */
}
```

**Practical Example:**

```css
/* Card flip with zoom */
.card {
  transition: transform 0.5s ease;
}

.card:hover {
  transform: translateY(-10px) scale(1.05) rotate(2deg);
}
```

## üéØ Transform Origin

Controls the point around which transforms occur.

```css
/* Default is center (50% 50%) */
.box {
  transform-origin: center;
  transform-origin: 50% 50%;
}

/* Keywords */
.box {
  transform-origin: top left;
  transform-origin: bottom right;
  transform-origin: center;
}

/* Precise values */
.box {
  transform-origin: 0 0; /* Top-left corner */
  transform-origin: 100% 100%; /* Bottom-right corner */
  transform-origin: 50px 50px; /* 50px from top-left */
}
```

**Practical Examples:**

```css
/* Rotate from corner */
.door {
  transform-origin: left center;
  transform: rotateY(90deg);
}

/* Scale from top */
.dropdown {
  transform-origin: top center;
  transform: scaleY(0);
  transition: transform 0.3s;
}

.dropdown.open {
  transform: scaleY(1);
}
```

## üé® 3D Transforms

### perspective

Creates 3D space for child elements.

```css
/* On parent container */
.container {
  perspective: 1000px; /* Distance from viewer */
}

/* Or on element itself */
.box {
  transform: perspective(1000px) rotateY(45deg);
}
```

### translate3d()

Move elements in 3D space.

```css
.box {
  transform: translate3d(50px, 100px, -200px); /* X, Y, Z */
}

/* Individual axes */
.box {
  transform: translateZ(100px); /* Towards viewer */
  transform: translateZ(-100px); /* Away from viewer */
}
```

### rotate3d()

Rotate in 3D space.

```css
/* Rotate around X-axis (horizontal flip) */
.box {
  transform: rotateX(45deg);
}

/* Rotate around Y-axis (vertical flip) */
.box {
  transform: rotateY(45deg);
}

/* Rotate around Z-axis (same as 2D rotate) */
.box {
  transform: rotateZ(45deg);
}

/* Rotate around custom axis */
.box {
  transform: rotate3d(1, 1, 0, 45deg); /* X, Y, Z, angle */
}
```

### scale3d()

Scale in 3D space.

```css
.box {
  transform: scale3d(2, 2, 2); /* X, Y, Z */
}

/* Individual axes */
.box {
  transform: scaleZ(2); /* Z-axis only */
}
```

### Example 1: Card Flip

```css
.card-container {
  perspective: 1000px;
}

.card {
  width: 300px;
  height: 200px;
  position: relative;
  transform-style: preserve-3d;
  transition: transform 0.6s;
}

.card:hover {
  transform: rotateY(180deg);
}

.card-front,
.card-back {
  position: absolute;
  width: 100%;
  height: 100%;
  backface-visibility: hidden;
}

.card-back {
  transform: rotateY(180deg);
}
```

### Example 2: 3D Cube

```css
.cube {
  width: 200px;
  height: 200px;
  position: relative;
  transform-style: preserve-3d;
  animation: rotate-cube 10s infinite linear;

.cube-face {
  position: absolute;
  width: 200px;
  height: 200px;
  opacity: 0.9;
}

.cube-front {
  transform: translateZ(100px);
}
.cube-back {
  transform: translateZ(-100px) rotateY(180deg);
}
.cube-right {
  transform: rotateY(90deg) translateZ(100px);
}
.cube-left {
  transform: rotateY(-90deg) translateZ(100px);
}
.cube-top {
  transform: rotateX(90deg) translateZ(100px);

.cube-bottom {
  transform: rotateX(-90deg) translateZ(100px);

@keyframes rotate-cube {
  from {
    transform: rotateX(0) rotateY(0);

    transform: rotateX(360deg) rotateY(360deg);
```

## üé® Additional Transform Properties

### transform-style

Controls whether children live in 3D space.

```css
.parent {
  transform-style: flat; /* Default: children in 2D */
  transform-style: preserve-3d; /* Children in 3D */
}
```

### backface-visibility

Controls whether back of element is visible.

```css
.card {
  backface-visibility: visible; /* Default */
  backface-visibility: hidden; /* Hide back side */
}
```

## ‚úÖ Best Practices

1. **Use transform for animations**

   ```css
   /* Good: Hardware accelerated */
   .element {
     transform: translateX(100px);
     transition: transform 0.3s;
   }

   /* Avoid: Causes reflow */
   .element {
     left: 100px;
     transition: left 0.3s;
   }
   ```

2. **Use translate3d for better performance**

   ```css
   /* Better: Forces GPU acceleration */
   .element {
     transform: translate3d(50px, 0, 0);
   }

   /* Good, but less optimized */
   .element {
     transform: translateX(50px);
   }
   ```

3. **Set transform-origin before rotating**

   ```css
   .element {
     transform-origin: left center;
     transform: rotate(45deg);
   }
   ```

4. **Use perspective on parent container**

   ```css
   /* Correct */
   .container {
     perspective: 1000px;
   }
   .child {
     transform: rotateY(45deg);
   }
   ```

## ‚ö†Ô∏è Common Pitfalls

### ‚ùå Transform Order Matters

```css
/* Different results! */
.box1 {
  transform: rotate(45deg) translate(100px, 0);
}

.box2 {
  transform: translate(100px, 0) rotate(45deg);
}
```

### ‚ùå Z-index with Transforms

```css
/* Transforms create new stacking context */
.element {
  transform: translateZ(0);
  z-index: 10; /* May not work as expected */
}
```

### ‚ùå Text Rendering

```css
/* Bad: Blurry text */
.element {
  transform: scale(1.5);
}

/* Better: Scale container, not text */
.element {
  font-size: 1.5em;
}
```
