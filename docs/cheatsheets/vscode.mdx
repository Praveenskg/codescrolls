#  VSCode Cheatsheet

A comprehensive cheat sheet to help you boost productivity with Visual Studio Code.

---

##  Basic Usage

```bash
# Launch VSCode
code .           # open current folder
code myfile.js   # open specific file

# Create new file/project
File > New File
File > Open Folder

# Quick Open
Ctrl/Cmd + P     # quickly open file by name
```

---

##  File Navigation

```bash
Ctrl/Cmd + P          # Quick open
Ctrl/Cmd + Shift + E  # Explorer
Ctrl/Cmd + B          # Toggle sidebar
Ctrl/Cmd + Tab        # Switch editor tab
```

---

## Settings

```bash
Ctrl/Cmd + ,         # Open settings
Ctrl/Cmd + Shift + P # Command palette
> Preferences: Open Settings (JSON)  # to edit manually
```

**Common Settings (settings.json)**

```json
{
  "editor.fontSize": 16,
  "editor.tabSize": 2,
  "editor.formatOnSave": true,
  "files.autoSave": "onFocusChange",
  "workbench.colorTheme": "One Dark Pro",
  "terminal.integrated.fontFamily": "Fira Code"
}
```

---

##  Common Shortcuts

```bash
Ctrl/Cmd + Shift + P      # Command palette
Ctrl/Cmd + /              # Toggle line comment
Alt + Up/Down             # Move line
Shift + Alt + Down        # Copy line
Ctrl/Cmd + D              # Select next word match
Ctrl/Cmd + L              # Select entire line
Ctrl/Cmd + Shift + L      # Select all occurrences
Ctrl/Cmd + Shift + K      # Delete line
Ctrl/Cmd + Enter          # Insert line below
Ctrl/Cmd + Shift + Enter  # Insert line above
```

---

##  Search & Replace

```bash
Ctrl/Cmd + F       # Find
Ctrl/Cmd + H       # Replace
Ctrl/Cmd + Shift + F # Global find
```

---

##  Debugging

```bash
F5                 # Start debugging
Shift + F5         # Stop debugging
F9                 # Toggle breakpoint
F10                # Step over
F11                # Step into
```

---

##  Git Integration

```bash
Ctrl/Cmd + Shift + G   # Open Git panel

# In command palette
> Git: Clone
> Git: Commit
> Git: Push
> Git: Pull

# Inline commands
Click ✔️ icon for commit
Click ↕️ icon to sync
```

---

##  Integrated Terminal

```bash
Ctrl/Cmd + `          # Toggle terminal
Ctrl/Cmd + Shift + `  # New terminal
Ctrl/Cmd + C          # Kill terminal process
```

---

##  Editor Management

```bash
Ctrl/Cmd + \                 # Split editor
Ctrl/Cmd + 1/2/3             # Focus on editor group
Ctrl/Cmd + Shift + [ or ]    # Switch tabs
```

---

##  Refactoring

```bash
F2                # Rename symbol
Ctrl/Cmd + .      # Quick fix / refactor
```

---

##  Markdown Support

```bash
Ctrl/Cmd + Shift + V   # Open Markdown preview
Ctrl/Cmd + K V         # Open Markdown preview to the side
```

---

## Remote Development

- Install **Remote - SSH** extension  
- Command Palette → `Remote-SSH: Connect to Host...`  
- Use with WSL, Containers, or Remote Machines

---

##  Emmet Abbreviations (HTML/CSS)

Example:  

```html
div.container>ul>li*5
```

Expands to:

```html
<div class="container">
  <ul>
    <li></li>
    <li></li>
    <li></li>
    <li></li>
    <li></li>
  </ul>
</div>
```
