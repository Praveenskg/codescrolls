#  VSCode Cheatsheet

A comprehensive cheat sheet to help you boost productivity with Visual Studio Code.

---

##  Basic Usage

```bash
# Launch VSCode
code .           # open current folder
code myfile.js   # open specific file

# Create new file/project
File > New File
File > Open Folder

# Quick Open
Ctrl/Cmd + P     # quickly open file by name
```

---

##  File Navigation

```bash
Ctrl/Cmd + P          # Quick open
Ctrl/Cmd + Shift + E  # Explorer
Ctrl/Cmd + B          # Toggle sidebar
Ctrl/Cmd + Tab        # Switch editor tab
```

---

## Settings

```bash
Ctrl/Cmd + ,         # Open settings
Ctrl/Cmd + Shift + P # Command palette
> Preferences: Open Settings (JSON)  # to edit manually
```

**Common Settings (settings.json)**

```json
{
  "editor.fontSize": 16,
  "editor.tabSize": 2,
  "editor.formatOnSave": true,
  "files.autoSave": "onFocusChange",
  "workbench.colorTheme": "One Dark Pro",
  "terminal.integrated.fontFamily": "Fira Code"
}
```

##  Common Shortcuts

```bash
Ctrl/Cmd + Shift + P      # Command palette
Ctrl/Cmd + /              # Toggle line comment
Alt + Up/Down             # Move line
Shift + Alt + Down        # Copy line
Ctrl/Cmd + D              # Select next word match
Ctrl/Cmd + L              # Select entire line
Ctrl/Cmd + Shift + L      # Select all occurrences
Ctrl/Cmd + Shift + K      # Delete line
Ctrl/Cmd + Enter          # Insert line below
Ctrl/Cmd + Shift + Enter  # Insert line above
```

##  Emmet Abbreviations (HTML/CSS)

Example:  

```html
div.container>ul>li*5
```

Expands to:

```html
<div class="container">
  <ul>
    <li></li>
    <li></li>
    <li></li>
    <li></li>
    <li></li>
  </ul>
</div>
```

## Advanced Shortcuts

### Multi-Cursor Editing

```bash
Alt + Click                    # Add cursor at click position
Ctrl/Cmd + Alt + Up/Down       # Add cursor above/below
```

## Advanced Emmet

### HTML Emmet

```html
<!-- Basic -->
div.container>ul>li*5

<!-- Advanced -->
div.container>header>nav>ul>li*4>a[href="#$"]{Link $}+main>section*3>h2{Section $}+p{Lorem ipsum}

<!-- Bootstrap -->
.container>.row>.col-md-6*2

<!-- React -->
div.container>h1{Title}+p{Description}+button.btn.btn-primary{Click Me}
```

### CSS Emmet

```css
/* Basic */
m10 → margin: 10px;
p20 → padding: 20px;
w100 → width: 100px;
h50 → height: 50px;

/* Advanced */
m10-20 → margin: 10px 20px;
p10-20-30 → padding: 10px 20px 30px;
w100+h50 → width: 100px; height: 50px;

/* Flexbox */
fx → display: flex;
fxc → display: flex; justify-content: center;
fxa → display: flex; align-items: center;
```

### Performance Optimization

```json
{
  "files.watcherExclude": {
    "**/.git/objects/**": true,
    "**/node_modules/**": true,
    "**/dist/**": true,
    "**/build/**": true
  },
  "search.exclude": {
    "**/node_modules": true,
    "**/bower_components": true,
    "**/dist": true,
    "**/build": true,
    "**/.git": true
  }
}
```
