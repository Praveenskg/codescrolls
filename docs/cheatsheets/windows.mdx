# Windows Cheatsheet

A complete reference for Windows PowerShell and Command Prompt (CMD) commands —
from file management to system administration, networking, and beyond.

---

## File & Directory Navigation

### PowerShell

```powershell
# Show current directory
Get-Location
pwd                # alias

# List contents
Get-ChildItem
ls                 # alias
dir                # alias
ls -Force          # include hidden
ls -Recurse        # recursive

# Change directory
Set-Location C:\Path\To\Dir
cd C:\Path\To\Dir  # alias
cd ~               # home directory
cd ..              # up one level

# Create or remove directories
New-Item -ItemType Directory -Path "NewFolder"
mkdir NewFolder    # alias
Remove-Item "Folder" -Recurse -Force
rmdir Folder       # empty directory
```

### CMD

```cmd
# Show current directory
cd

# List contents
dir
dir /a             # include hidden
dir /s             # recursive

# Change directory
cd C:\Path\To\Dir
cd %USERPROFILE%   # home directory
cd ..              # up one level

# Create or remove directories
mkdir NewFolder
md NewFolder       # alias
rmdir Folder       # empty directory
rmdir /s /q Folder # non-empty, quiet
```

## File Operations

### PowerShell

```powershell
# Create a file
New-Item -ItemType File -Path "file.txt"
echo "content" > file.txt

# View file contents
Get-Content file.txt
cat file.txt       # alias
gc file.txt        # alias
gc file.txt -Tail 10        # last 10 lines
gc file.txt -Wait           # live updates

# Copy, move, rename
Copy-Item source.txt dest.txt
cp source.txt dest.txt      # alias
Move-Item file.txt newname.txt
mv file.txt newname.txt     # alias
Remove-Item file.txt
rm file.txt                 # alias

# Search inside files
Select-String "text" file.txt
sls "text" file.txt         # alias
sls "text" -Path *.txt -Recurse
```

### CMD

```cmd
# Create a file
echo content > file.txt
type nul > file.txt  # empty file

# View file contents
type file.txt
more file.txt        # paginated

# Copy, move, rename
copy source.txt dest.txt
move file.txt newname.txt
del file.txt         # delete

# Search inside files
findstr "text" file.txt
findstr /s /i "text" *.txt  # recursive, case-insensitive
```

## Permissions & Security

### PowerShell

```powershell
# View file permissions (ACL)
Get-Acl file.txt | Format-List

# Set permissions
$acl = Get-Acl file.txt
$rule = New-Object System.Security.AccessControl.FileSystemAccessRule("User","FullControl","Allow")
$acl.SetAccessRule($rule)
Set-Acl file.txt $acl

# Take ownership
takeown /f file.txt
icacls file.txt /grant User:F

# View execution policy
Get-ExecutionPolicy

# Set execution policy
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
Set-ExecutionPolicy Bypass -Scope Process
```

### CMD

```cmd
# View/modify permissions
icacls file.txt
icacls file.txt /grant User:F  # full control
icacls file.txt /deny User:R   # deny read

# Take ownership
takeown /f file.txt
takeown /r /f folder           # recursive
```

```cmd
taskkill /IM notepad.exe

# Start a process
start notepad.exe
start "" "C:\Program Files\app.exe"
```

## Advanced File Operations

### Text Processing (PowerShell)

```powershell
# Sort and filter
Get-Content file.txt | Sort-Object

Get-Content file.txt | Sort-Object -Unique

Get-Content file.txt | Where-Object { $_ -notmatch "pattern" }

Get-Content file.txt | ForEach-Object { $_.Replace("old", "new") }

Get-Content file.txt | ForEach-Object { $_.Trim() }

Import-Csv data.csv | Select-Object Name, Email | Export-Csv filtered.csv

Get-Content data.json | ConvertFrom-Json
$data = @{Name="John"; Age=30}
$data | ConvertTo-Json | Out-File data.json
```

### Advanced Search (PowerShell)

```powershell
# Find files by criteria
Get-ChildItem -Path C:\ -Filter *.log -Recurse -ErrorAction SilentlyContinue
Get-ChildItem -Recurse | Where-Object { $_.Length -gt 100MB }
Get-ChildItem -Recurse | Where-Object { $_.LastWriteTime -gt (Get-Date).AddDays(-7) }

Get-ChildItem -Filter *.log -Recurse | ForEach-Object { $_.FullName }
```

## Registry Operations

### PowerShell

```powershell
# Navigate registry (like filesystem)
Set-Location HKCU:\Software
cd HKLM:\Software

# Read registry values
Get-ItemProperty -Path "HKCU:\Software\Microsoft\Windows\CurrentVersion"
Get-ItemPropertyValue -Path "HKCU:\Software\Microsoft\Windows\CurrentVersion\Explorer" -Name "ShellState"

# Create registry key
New-Item -Path "HKCU:\Software\MyApp"
New-ItemProperty -Path "HKCU:\Software\MyApp" -Name "Setting" -Value "Value" -PropertyType String

# Modify registry value
Set-ItemProperty -Path "HKCU:\Software\MyApp" -Name "Setting" -Value "NewValue"

# Delete registry key/value
Remove-Item -Path "HKCU:\Software\MyApp" -Recurse
Remove-ItemProperty -Path "HKCU:\Software\MyApp" -Name "Setting"

# Search registry
Get-ChildItem -Path HKCU:\Software -Recurse | Where-Object { $_.PSChildName -match "pattern" }
```

### CMD

```cmd
# Query registry
reg query "HKCU\Software\Microsoft\Windows\CurrentVersion"
reg query "HKLM\Software" /s       # recursive

# Add registry value
reg add "HKCU\Software\MyApp" /v Setting /t REG_SZ /d "Value"
reg add "HKCU\Software\MyApp" /v Number /t REG_DWORD /d 123

# Modify registry value
reg add "HKCU\Software\MyApp" /v Setting /t REG_SZ /d "NewValue" /f

# Delete registry key/value
reg delete "HKCU\Software\MyApp" /f
reg delete "HKCU\Software\MyApp" /v Setting /f

# Export/Import registry
reg export "HKCU\Software\MyApp" backup.reg
reg import backup.reg
```

## PowerShell Scripting

### Basic Scripting

```powershell
# Variables
$name = "John"
$age = 25
Write-Output "Hello $name, you are $age years old"
Write-Host "Hello $name" -ForegroundColor Green

# Arrays
$array = @(1, 2, 3, 4, 5)
$array[0]              # first element
$array[-1]             # last element
$array.Count           # length

# Hashtables (dictionaries)
$hash = @{
    Name = "John"
    Age = 30
    City = "New York"
}
$hash["Name"]
$hash.Age

# Conditional statements
if ($age -gt 18) {
    Write-Output "Adult"
} elseif ($age -eq 18) {
    Write-Output "Just turned adult"
} else {
    Write-Output "Minor"
}

# Comparison operators
# -eq, -ne, -gt, -ge, -lt, -le, -like, -notlike, -match, -notmatch

# Loops
foreach ($item in $array) {
    Write-Output "Item: $item"
}

for ($i = 0; $i -lt 5; $i++) {
    Write-Output "Number: $i"
}

while ($i -lt 10) {
    Write-Output $i
    $i++
}

# Functions
function Greet {
    param(
        [string]$Name,
        [int]$Age = 0
    )
    Write-Output "Hello $Name, you are $Age years old"
}
Greet -Name "John" -Age 30
```

### Advanced Scripting

```powershell
# Error handling
try {
    Get-Content "nonexistent.txt" -ErrorAction Stop
} catch {
    Write-Error "Error: $_"
} finally {
    Write-Output "Cleanup"
}

# Parameter validation
function ValidateAge {
    param(
        [Parameter(Mandatory=$true)]
        [ValidateRange(0, 120)]
        [int]$Age
    )
    Write-Output "Age is $Age"
}

# Pipeline and filters
Get-Process | Where-Object { $_.CPU -gt 10 } | Sort-Object CPU -Descending | Select-Object -First 5

# Custom objects
$obj = [PSCustomObject]@{
    Name = "John"
    Age = 30
    City = "New York"
}

# File monitoring
$watcher = New-Object System.IO.FileSystemWatcher
$watcher.Path = "C:\Temp"
$watcher.Filter = "*.txt"
$watcher.EnableRaisingEvents = $true
Register-ObjectEvent $watcher "Created" -Action {
    Write-Host "File created: $($Event.SourceEventArgs.Name)"
}

# Scheduled tasks
$action = New-ScheduledTaskAction -Execute "PowerShell.exe" -Argument "-File C:\script.ps1"
$trigger = New-ScheduledTaskTrigger -Daily -At 9am
Register-ScheduledTask -Action $action -Trigger $trigger -TaskName "MyTask"
```

## Advanced Networking

### Network Troubleshooting (PowerShell)

```powershell
# Connectivity tests
Test-Connection google.com -Count 4
Test-NetConnection google.com -Port 443
Test-NetConnection -ComputerName server -CommonTCPPort RDP

# Trace route
Test-NetConnection google.com -TraceRoute

# DNS testing
Resolve-DnsName google.com
Resolve-DnsName google.com -Type MX
Resolve-DnsName google.com -Server 8.8.8.8

# Port scanning
Test-NetConnection -ComputerName server -Port 443
1..1024 | ForEach-Object { Test-NetConnection -ComputerName server -Port $_ -WarningAction SilentlyContinue } | Where-Object { $_.TcpTestSucceeded }

# Network shares
Get-SmbShare
New-SmbShare -Name "SharedFolder" -Path "C:\Share" -FullAccess "Everyone"
Remove-SmbShare -Name "SharedFolder" -Force
```

### Firewall Management (PowerShell)

```powershell
# Firewall status
Get-NetFirewallProfile

# Enable/disable firewall
Set-NetFirewallProfile -Profile Domain,Public,Private -Enabled True
Set-NetFirewallProfile -Profile Domain,Public,Private -Enabled False

# Firewall rules
Get-NetFirewallRule | Where-Object { $_.Enabled -eq $true }
Get-NetFirewallRule -DisplayName "Rule Name"

# Add firewall rule
New-NetFirewallRule -DisplayName "Allow Port 8080" -Direction Inbound -Protocol TCP -LocalPort 8080 -Action Allow
New-NetFirewallRule -DisplayName "Block App" -Direction Outbound -Program "C:\app.exe" -Action Block

# Remove firewall rule
Remove-NetFirewallRule -DisplayName "Rule Name"

# Enable remote desktop
Set-ItemProperty -Path "HKLM:\System\CurrentControlSet\Control\Terminal Server" -Name "fDenyTSConnections" -Value 0
Enable-NetFirewallRule -DisplayGroup "Remote Desktop"
```

### Firewall Management (CMD)

```cmd
# Firewall status
netsh advfirewall show allprofiles

# Enable/disable firewall
netsh advfirewall set allprofiles state on
netsh advfirewall set allprofiles state off

# Add firewall rule
netsh advfirewall firewall add rule name="Allow Port 8080" dir=in action=allow protocol=TCP localport=8080
netsh advfirewall firewall add rule name="Block App" dir=out action=block program="C:\app.exe"

# Remove firewall rule
netsh advfirewall firewall delete rule name="Rule Name"

# Show firewall rules
netsh advfirewall firewall show rule name=all
```

## Remote Management

### PowerShell Remoting

```powershell
# Enable PS Remoting
Enable-PSRemoting -Force

# Connect to remote computer
Enter-PSSession -ComputerName Server01
Enter-PSSession -ComputerName Server01 -Credential (Get-Credential)

# Run command on remote computer
Invoke-Command -ComputerName Server01 -ScriptBlock { Get-Process }
Invoke-Command -ComputerName Server01,Server02 -ScriptBlock { Get-Service }

# Copy files to remote
Copy-Item -Path "C:\file.txt" -Destination "C:\destination\" -ToSession $session

# Persistent sessions
$session = New-PSSession -ComputerName Server01
Invoke-Command -Session $session -ScriptBlock { Get-Process }
Remove-PSSession $session
```

### Remote Desktop

```powershell
# Enable RDP via PowerShell
Set-ItemProperty -Path "HKLM:\System\CurrentControlSet\Control\Terminal Server" -Name "fDenyTSConnections" -Value 0
Enable-NetFirewallRule -DisplayGroup "Remote Desktop"

# Connect via RDP
mstsc /v:server.domain.com
mstsc /v:192.168.1.100 /admin  # console session
```

```cmd

psexec \\server -u user -p password cmd

shutdown /m \\server /r /t 0
```

## System Maintenance

### Windows Update (PowerShell)

```powershell
# Check for updates (requires PSWindowsUpdate module)
Install-Module PSWindowsUpdate -Force
Get-WindowsUpdate
Install-WindowsUpdate -AcceptAll -AutoReboot

# View update history
Get-HotFix
Get-HotFix | Sort-Object InstalledOn -Descending
```

### System File Checker

```cmd
# Scan system files
sfc /scannow

# Scan but don't repair
sfc /verifyonly

# Scan offline Windows
sfc /scannow /offbootdir=C:\ /offwindir=C:\Windows
```

### DISM (Deployment Image Servicing)

```cmd
# Check image health
DISM /Online /Cleanup-Image /CheckHealth

# Scan image
DISM /Online /Cleanup-Image /ScanHealth

# Restore image
DISM /Online /Cleanup-Image /RestoreHealth

# Cleanup component store
DISM /Online /Cleanup-Image /StartComponentCleanup
```
